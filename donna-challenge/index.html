The Complete Guide to Claude Code

המדריך המלא לקלוד קוד. קח את הזמן, יש לך שבוע שלם. עבור שלב שלב, אין לחץ.

הסרטון יעלה בקרוב

🧠 שלב 1: מה זה Claude Code ולמה הוא שונה

כנראה השתמשת ב-ChatGPT או ב-Google Gemini. הם צ'אטבוטים: אתה שואל, הם עונים, הם שוכחים. כל שיחה מתחילה מאפס.

Claude Code זה משהו אחר לגמרי. זה לא צ'אטבוט. זה סביבת עבודה שלמה.

שלושה הבדלים מרכזיים:

1. הוא זוכר. יש קובץ בשם CLAUDE.md שהוא קורא לפני כל שיחה. מה שתכתוב שם, הוא ידע. תמיד.

2. הוא מתחבר. אפשר לחבר אותו למייל שלך, ליומן, לדרייב, לאסנה. הוא באמת קורא את המיילים שלך. לא מיילים דמיוניים.

3. הוא עובד לבד. אפשר להגדיר משימות שרצות לפי לוח זמנים. "כל בוקר ב-9:15, תקרא את המיילים שלי ותכין לי סיכום." הוא עושה את זה. אוטומטית.

במשפט אחד: ChatGPT מדבר איתך. Claude Code עובד בשבילך.

💻 שלב 2: התקנת Claude Desktop

Claude Code רץ בתוך אפליקציה שנקראת Claude Desktop. לא בדפדפן, לא בטרמינל. אפליקציה על המחשב.

Mac:

  1. לך ל-claude.ai/download
  2. הורד את קובץ ה-.dmg
  3. פתח אותו וגרור את Claude לתיקיית Applications

Windows:

  1. לך ל-claude.ai/download
  2. הורד את ה-installer, לחץ Next עד הסוף

Windows בלבד: חובה להתקין Git

Claude Code משתמש ב-Git ברקע. בלי Git, Claude Code לא יעבוד על Windows. זה חינמי ולוקח דקה.

  1. לך ל-git-scm.com/download/win
  2. הורד והתקן (Next Next Finish)
  3. חשוב: וודא שהאפשרות "Add Git to PATH" מסומנת (היא מסומנת כברירת מחדל)

Mac לא צריך שום דבר נוסף.

אחרי ההתקנה, פתח את Claude Desktop בפעם הראשונה. הוא יבקש ממך להתחבר או ליצור חשבון. עקוב אחרי ההוראות על המסך.

💳 שלב 3: מנוי Pro ($20/חודש)

הגרסה החינמית מוגבלת מדי לצרכים שלנו. מנוי Pro פותח את כל מה שצריך באתגר: Skills, Scheduled Tasks, Connectors.

איך לשדרג:

  1. פתח את Claude Desktop
  2. לחץ על האייקון של הפרופיל שלך (פינה שמאלית תחתונה)
  3. בחר Upgrade או Subscribe
  4. בחר את תוכנית Pro ב-$20/חודש
  5. הכנס כרטיס אשראי ואשר

על הקרדיטים

תוכנית Pro מגיעה עם כמות מוגבלת של קרדיטים (שימושים). ככל שתשתמש יותר, תבחין שהם נגמרים מהר יותר. כשזה קורה, יש תוכנית Max ב-$100/חודש עם הרבה יותר קרדיטים. אבל לאתגר ולהתחלה, Pro מספיק.

מומלץ להשתמש במודל Sonnet כברירת מחדל כדי לחסוך קרדיטים. נסביר על המודלים בשלב 8.

🔀 שלב 4: שלושת המצבים (Chat, Cowork, Code)

ב-Claude Desktop יש 3 טאבים למעלה. זה הדבר הכי חשוב להבין לפני שמתחילים:

💬
Chat

זה המקבילה של ChatGPT. שאלה, תשובה. בלי גישה לקבצים או כלים.

🤝
Cowork

ממשק נוח יותר לעבודה עם קבצים. Claude רואה ויכול לערוך. טוב לעריכת מסמכים.

⚡
Code

הטרמינל הידידותי. כאן Claude מריץ Skills, Scheduled Tasks, מתחבר לכלים. כאן דונה חיה. כאן אנחנו עובדים.

כלל: תמיד לעבוד מ-Code. לא Chat, לא Cowork. Code.

📁 שלב 5: Claude עובד עם תיקיות

זה קונספט חשוב שצריך להבין. Claude Code לא עובד "באוויר". הוא מתחבר לתיקייה ספציפית על המחשב שלך.

לפני כל סשן עבודה, אתה בוחר תיקייה (File > Open Folder). Claude קורא את הקבצים בתיקייה הזו, כותב אליה, ומשתמש בקובץ CLAUDE.md שנמצא בה.

זו הסיבה שדונה "חיה" בתיקייה ספציפית (donna-starter-kit). כשתפתח את התיקייה הזו ב-Claude Code, דונה תדע מי היא ומה היא יודעת לעשות. כשתפתח תיקייה אחרת, Claude לא יכיר את דונה.

בקיצור:

תיקייה = פרויקט. תיקייה אחרת = פרויקט אחר. CLAUDE.md בתיקייה = ההוראות שלך ל-Claude באותו פרויקט.

🗂️ שלב 6: הכרת הסיידבר של Claude Desktop

בצד שמאל של Claude Desktop (בטאב Code) יש סיידבר. כדאי להכיר מה יש בו:

New session - פותח שיחה חדשה. דונה לא זוכרת שיחות קודמות, אבל כן קוראת את CLAUDE.md מחדש. כל שיחה מתחילה "נקייה" עם הזיכרון מהקובץ.

Search - חיפוש בשיחות קודמות. שימושי כשאתה רוצה למצוא משהו שביקשת בעבר.

Scheduled - כאן רואים ומגדירים משימות מתוזמנות שרצות אוטומטית (נלמד ביום 5).

Customize - הגדרות, חיבור כלים חיצוניים (Connectors), והגדרות מודל. נכנס לזה בשלב הבא.

⚙️ שלב 7: הגדרות חשובות

כדאי להכיר את ההגדרות לפני שמתחילים לעבוד.

איך להגיע להגדרות:

לחצו על Customize בסיידבר (בצד שמאל), או על אייקון הפרופיל בפינה השמאלית התחתונה.

הגדרה חשובה: בחירת מודל

Claude מגיע עם כמה מודלים (גרסאות) שונים. המודל קובע כמה חכם וכמה קרדיטים הוא צורך. מומלץ להגדיר את Sonnet כברירת מחדל כדי לחסוך קרדיטים. נסביר על ההבדלים בשלב הבא.

את בחירת המודל אפשר לשנות גם מתוך שיחה: לחצו על שם המודל בפינה העליונה של חלון השיחה.

🤖 שלב 8: המודלים של Claude

Claude מגיע בכמה "גרסאות" שנקראות מודלים. כל מודל שונה ברמת החוכמה, המהירות, וכמות הקרדיטים שהוא צורך:

🐇
Haiku

מהיר, קל, זול. טוב למשימות פשוטות כמו תרגום, סיכום קצר, או שאלה מהירה.

⚡
Sonnet

מאוזן. מומלץ לאתגר. חכם מספיק לרוב המשימות, חוסך קרדיטים.

🧠
Opus

הכי חזק. צורך יותר קרדיטים. שמרו לניתוח מורכב או עבודה שדורשת חשיבה מעמיקה.

איך לשנות מודל:

הקלידו /model בתוך שיחה, או לחצו על שם המודל בפינה העליונה של חלון השיחה, או שנו בהגדרות (Customize).

כדי לחסוך קרדיטים, השתמשו ב-Sonnet כברירת מחדל.

Sonnet חכם מספיק ל-90% מהמשימות. תעברו ל-Opus רק כשצריכים ניתוח עמוק או כתיבה מורכבת.

🔐 שלב 9: מצבי אישורים

כשClaude רוצה לעשות משהו (לערוך קובץ, להריץ פקודה), הוא מבקש אישור. יש 4 מצבים:

✋ Ask permissions

Claude שואל לפני כל שינוי. הכי בטוח. טוב למתחילים שרוצים לראות מה קורה לפני שמאשרים.

</> Auto accept edits

Claude עורך קבצים בלי לשאול, אבל עדיין שואל לפני הרצת פקודות. אמצע הדרך.

📋 Plan mode

Claude מתכנן לפני שעושה. מציג תכנית ומחכה לאישור. טוב למשימות גדולות שרוצים לראות תכנית לפני ביצוע.

⚠️ Bypass permissions

Claude עושה הכל בלי לשאול. הכי מהיר. מומלץ למי שכבר מרגיש בנוח עם Claude ורוצה שהעבודה תזרום בלי עצירות.

המלצה לאתגר: תתחילו עם Auto accept edits. אחרי כמה ימים שתרגישו בנוח, עברו ל-Bypass permissions.

איפה משנים: לחצו על האייקון בפינה השמאלית התחתונה של שורת הקלט (ליד כפתור השליחה).

⚡ שלב 10: פקודות שימושיות (Slash Commands)

ב-Claude Code יש פקודות מובנות שמתחילות ב-/. כמו קיצורי דרך. הנה הכי חשובות:

/compact כשהשיחה מתארכת ו-Claude מתחיל להיות איטי. דוחס את השיחה לסיכום וממשיך חד.

/clear מתחיל שיחה חדשה לגמרי. Claude שוכח הכל (אבל עדיין קורא את CLAUDE.md).

/model מחליף מודל באמצע שיחה. לא צריך לפתוח הגדרות.

/help מראה את כל הפקודות הזמינות.

תנסה עכשיו:

/help

🎯 שלב 11: איך לבקש מ-Claude דברים (Prompting)

איך אתה מבקש משנה את מה שתקבל. הנה דוגמה:

בקשה עמומה (לא טוב)

make me a calculator

תוצאה: Claude ישאל שאלות הבהרה, או יבנה משהו גנרי שלא מתאים לך.

בקשה ברורה (מעולה)

Build me a simple HTML profit calculator. I enter cost and selling price, it shows margin as %. Make it clean and modern. Save as calculator.html

תוצאה: כלי עובד ב-30 שניות.

הכלל: ספר ל-Claude מה אתה רוצה, באיזה פורמט, ואיפה לשמור.

תנסה עכשיו. בנה מחשבון:

Build me a simple HTML profit calculator. I enter the product cost and selling price, and it shows me the profit margin as a percentage. Make it clean and modern looking. Save it as calculator.html

Claude יבנה את הכלי וישמור אותו. תפתח את הקובץ בדפדפן ותראה מחשבון עובד.

עכשיו תנסה דשבורד:

Build me a beautiful dark-mode dashboard that shows today's date, a to-do list I can add items to, and a motivational quote. Save as dashboard.html

שחק עם זה. תנסה דברים. אי אפשר לשבור כלום. זה הזמן להתנסות.

🎨 שלב 12: תראו מה אפשר לעשות (ותנסו בעצמכם)

בסרטון אני מראה דוגמאות חיות של דברים שבניתי עם Claude. דשבורדים, דפי נחיתה, כלים, מחשבונים. הכל בפקודה אחת, בלי קוד.

אחרי שתראו את הסרטון, תנסו בעצמכם. הנה כמה רעיונות:

Build me a beautiful dark-mode dashboard that shows today's date, a to-do list I can add items to, and a motivational quote. Save as dashboard.html
Build me a modern landing page for a product called "SuperJuice". It's a health drink. Include a hero section, 3 benefits, testimonials, and a buy button. Save as landing-page.html
Build me a simple expense tracker where I can add expenses with a name, amount, and category. Show a total at the bottom and a pie chart. Save as expenses.html

אין חובה לעשות את זה. אבל אם תנסו, תבינו כמה Claude חזק. וזה עוד לפני שלימדנו אותו מי אתם ומה העסק שלכם.

🔑 שלב 13: טיפ זהב. ייבוא זיכרון מ-ChatGPT

עוברים מ-ChatGPT? אל תתחילו מאפס. Claude יכול לייבא את הזיכרונות שצברתם ב-ChatGPT, כך שהוא כבר ידע דברים עליכם מהרגע הראשון.

איך עושים את זה:

  1. ב-ChatGPT: בקשו ממנו לייצא את הזיכרון שלו. הוא יפרט את כל מה שהוא זוכר עליכם.
  2. העתיקו את הטקסט שהוא מחזיר.
  3. ב-Claude Desktop: היכנסו להגדרות (Customize) > Import memory > הדביקו את הטקסט.

מדריך מפורט: לחצו כאן למדריך המלא

שימו לב: זו תכונה ניסיונית. היא עובדת הכי טוב עבור העדפות עבודה ומידע מקצועי, פחות עבור פרטים אישיים. לא חייבים לעשות את זה, אבל זה חוסך זמן.

מתוך סדרת ההאקים שלי: Hack #1: Import Your Memory in 60 Seconds

🎓 שלב 14: טיפ זהב. פרומפט המאמן

יש טריק שגיליתי שהופך את Claude למאמן אישי. אתה מדביק פרומפט מיוחד בתחילת פרויקט, ו-Claude ממשיך לעבוד כרגיל, אבל מזמן לזמן זורק טיפים שעוזרים לך ללמוד.

למשל, אחרי שמסיימים משימה הוא עשוי להגיד: "כדאי לתעד את ההחלטה הזו. כדי שלא נבזבז שוב זמן על אותה שאלה." או אחרי שיחה ארוכה: "השיחה ארוכה, כדאי לדחוס אותה כדי לחסוך קונטקסט."

הוא לא מפסיק לעבוד ומתחיל ללמד. הוא עובד ומלמד במקביל. רק כשיש הזדמנות ללמידה ברורה.

חדשות טובות

כבר הטמענו את הפרומפט הזה בתוך ה-CLAUDE.md של דונה. כלומר, דונה כבר מגיעה עם "מצב מאמן" מובנה. לא צריך לעשות כלום נוסף.

📱 שלב 15: אפליקציות נוספות

Claude Desktop זה לא הכלי היחיד. כדאי לדעת שיש גם אפליקציות נוספות:

אפליקציית מובייל (iOS / Android) - מצב Chat בלבד. שימושי לשאלות מהירות בדרך. אין בו Code tab, אז אי אפשר להריץ דונה משם.

תוסף לדפדפן - Claude יכול לעבוד ישירות עם דפי אינטרנט שאתם גולשים בהם. שימושי לסיכומים ומחקר.

עבודה עם גיליונות - Claude יכול לקרוא ולנתח קבצי Excel ו-CSV. שימושי לניתוח נתונים ודוחות.

באתגר אנחנו עובדים רק מ-Claude Desktop (טאב Code). אבל כדאי לדעת שיש גם אפשרויות אחרות שתוכלו לחקור אחרי האתגר.

🟣 שלב 16: התקנת Obsidian

Obsidian זו אפליקציה חינמית שמציגה קבצי טקסט בצורה ויזואלית. אנחנו משתמשים בה כדי לראות את הזיכרון של דונה. כשדונה כותבת סיכום, מתעדת החלטה, או שומרת מידע על אנשים, Obsidian מציג את הכל בצורה מסודרת עם קשרים ביניהם.

  1. לך ל-obsidian.md והורד (חינמי, Mac/Windows/Linux)
  2. התקן כמו כל אפליקציה

נחבר את Obsidian אחרי שנוריד את ה-Starter Kit בשלב הבא.

📦 שלב 17: הורדת ה-Starter Kit

ה-Starter Kit זו תיקייה עם כל מה שדונה צריכה: זהות, זיכרון, סקילס, טמפלייטים, ומדריכים.

  1. הורד את ה-Starter Kit: לחץ כאן להורדה
  2. פתח את קובץ ה-ZIP (לחיצה כפולה)
  3. שים את התיקייה donna-starter-kit במקום נגיש (שולחן עבודה או Documents). לא בתוך תיקיות מקוננות.

מה בפנים:

donna-starter-kit/
  CLAUDE.md         <-- הזהות של דונה. נמלא ביום 2
  brain/            <-- הזיכרון של דונה (Obsidian)
  .claude/skills/  <-- פקודות שדונה יודעת (יום 4)
  templates/       <-- פורמטים לדוחות

  samples/         <-- דוגמאות כתיבה שלך
  output/          <-- כל מה שדונה יוצרת

אל תערוך כלום עדיין. ביום 1 נפתח ונתחיל.

🔗 שלב 18: חיבור הכל

עכשיו מחברים את הכל: Claude Code + Obsidian, שניהם על אותה תיקייה.

ב-Claude Desktop:

  1. פתח Claude Desktop
  2. עבור לטאב Code
  3. File > Open Folder > בחר את donna-starter-kit
  4. בדוק שזה עובד:
What files do you see in this folder?

Claude צריך לראות: CLAUDE.md, brain/, templates/, guides/ ועוד.

ב-Obsidian:

Claude Code ו-Obsidian שניהם מסתכלים על אותם קבצים. כשדונה כותבת דוח בוקר, הוא מופיע אוטומטית ב-Obsidian. שני חלונות, אותו מקום.

💡 טיפים חשובים

  • תמיד תעבוד מטאב Code. לא Chat, לא Cowork.
  • השתמשו ב-Sonnet כברירת מחדל כדי לחסוך קרדיטים. תעברו ל-Opus רק כשצריכים.
  • אם Claude שואל שאלות הבהרה, תענו לו. הוא מנסה להבין מה אתם רוצים.
  • אם משהו לא עובד, תנסחו מחדש. "Build me X that does Y and save as Z" עובד כמעט תמיד.
  • אי אפשר לשבור כלום. תנסו, תשחקו, תתנסו.

  • ה-Starter Kit צריך להיות במקום נגיש. לא בתוך 5 תיקיות מקוננות.
  • Obsidian ו-Claude Code פותחים את אותה תיקייה (donna-starter-kit).

❓ שאלות נפוצות

מה ההבדל בין Claude ל-ChatGPT?
ChatGPT הוא צ'אטבוט: שאלה, תשובה, שוכח. כל שיחה מתחילה מאפס. Claude Code הוא סביבת עבודה שלמה: הוא זוכר מידע דרך קובץ CLAUDE.md, מתחבר לכלים אמיתיים כמו מייל ויומן, יכול לבנות כלים ולשמור קבצים, ואפילו לרוץ אוטומטית לפי לוח זמנים. בקיצור, ChatGPT מדבר איתך. Claude Code עובד בשבילך.
למה צריך Obsidian?
Obsidian הוא "החלון" שלכם לזיכרון של דונה. דונה שומרת את כל מה שהיא לומדת בקבצי טקסט בתיקיית brain/. Obsidian מציג את הקבצים האלה בצורה ויזואלית, עם קשרים בין אנשים, פרויקטים, והחלטות. בלעדיו, דונה עדיין עובדת ושומרת הכל. אבל Obsidian מוסיף שכבה ויזואלית שעוזרת לכם לראות מה דונה יודעת ולנווט בין הנושאים.
כמה זה עולה?
$20 לחודש עבור מנוי Pro ל-Claude. זו העלות השוטפת היחידה. Obsidian חינמי, Git חינמי, ה-Starter Kit כלול באתגר. ככל שתשתמשו יותר ותרצו יותר קרדיטים, אפשר לשדרג ל-Max ב-$100/חודש. אבל Pro מספיק לאתגר ולהתחלה.
אני צריך לדעת לתכנת?
לא. בכלל לא. באתגר הזה לא כותבים שורת קוד אחת. אתם מדברים עם Claude בשפה רגילה (אנגלית), ומסבירים לו מה אתם רוצים. Claude עושה את העבודה הטכנית. זה כמו לתת הוראות לעובד חכם, לא כמו לכתוב תוכנה.
זה עובד על Mac ועל Windows?
כן, שניהם נתמכים. ההבדל היחיד: על Windows חובה להתקין גם Git (חינמי, התקנה של דקה). בלי Git, Claude Code לא יעבוד על Windows. על Mac, Git כבר מותקן ברקע ולא צריך לעשות כלום נוסף. כל שאר השלבים זהים.
מה זה CLAUDE.md?
קובץ טקסט פשוט שנמצא בתיקייה שלכם. Claude קורא אותו אוטומטית לפני כל שיחה חדשה. מה שכתוב בקובץ הזה, Claude ידע. זה "תעודת הזהות" של דונה, ההנחיות שלה, המידע על העסק שלכם, וסגנון העבודה המועדף. נמלא אותו יחד ביום 2 של האתגר.
מה זה Skills?
Skills הם קבצי טקסט עם הנחיות מפורטות שאומרות ל-Claude מה לעשות צעד אחר צעד. כמו מתכון. כותבים פעם אחת, Claude מבצע בכל פעם שמפעילים. למשל: /morning-briefing = "תקרא מיילים, תבדוק יומן, תכין סיכום." במקום להסביר מחדש בכל בוקר, פשוט מריצים את ה-Skill. נלמד לבנות Skills ביום 4.
מה זה Scheduled Tasks?
משימות שרצות אוטומטית לפי לוח זמנים שאתם מגדירים. למשל: "כל בוקר ב-9:15, תריץ את סיכום הבוקר." Claude יעשה את זה לבד, בלי שתצטרכו לפתוח את האפליקציה ולבקש. פשוט מגדירים פעם אחת, וזה רץ. נלמד את זה ביום 5 של האתגר.
מה ההבדל בין המודלים (Haiku, Sonnet, Opus)?
שלושת המודלים הם "גרסאות" שונות של Claude. Haiku הוא מהיר וזול, טוב לשאלות פשוטות. Sonnet מאוזן בין חוכמה לעלות, ומומלץ כברירת מחדל. Opus הוא הכי חזק וחכם, אבל צורך הכי הרבה קרדיטים. לאתגר, Sonnet מספיק ל-90% מהמשימות. תעברו ל-Opus רק כשצריכים ניתוח מורכב או כתיבה מעמיקה.
מה קורה כשנגמרים הקרדיטים?
בתוכנית Pro ($20/חודש), יש כמות מוגבלת של שימושים ביום. כשנגמרים, תצטרכו לחכות כמה שעות עד שהם מתאפסים, או לשדרג לתוכנית Max ($100/חודש) שמגיעה עם הרבה יותר קרדיטים. שימוש ב-Sonnet במקום Opus חוסך משמעותית. לאתגר, Pro אמור להספיק אם משתמשים ב-Sonnet כברירת מחדל.
יש מילון מונחים?
כן! הכנתי מילון אינטראקטיבי עם 50 מונחים של Claude שכולם שומעים אבל לא תמיד מבינים: tokens, context window, MCP, skills, ועוד. שמרו את זה בסימניות: Hack #13: 50 Claude Terms Explained
