The Complete Guide to Claude Code

המדריך המלא לקלוד קוד. קח את הזמן, יש לך שבוע שלם. עבור שלב שלב, אין לחץ.

הסרטון יעלה בקרוב

🧠 שלב 1: מה זה Claude Code ולמה הוא שונה

כנראה השתמשת ב-ChatGPT או ב-Google Gemini. הם צ'אטבוטים: אתה שואל, הם עונים, הם שוכחים. כל שיחה מתחילה מאפס.

Claude Code זה משהו אחר לגמרי. זה לא צ'אטבוט. זה סביבת עבודה שלמה.

שלושה הבדלים מרכזיים:

1. הוא זוכר. יש קובץ בשם CLAUDE.md שהוא קורא לפני כל שיחה. מה שתכתוב שם, הוא ידע. תמיד.

2. הוא מתחבר. אפשר לחבר אותו למייל שלך, ליומן, לדרייב, לאסנה. הוא באמת קורא את המיילים שלך. לא מיילים דמיוניים.

3. הוא עובד לבד. אפשר להגדיר משימות שרצות לפי לוח זמנים. "כל בוקר ב-9:15, תקרא את המיילים שלי ותכין לי סיכום." הוא עושה את זה. אוטומטית.

במשפט אחד: ChatGPT מדבר איתך. Claude Code עובד בשבילך.

💻 שלב 2: התקנת Claude Desktop

Claude Code רץ בתוך אפליקציה שנקראת Claude Desktop. לא בדפדפן, לא בטרמינל. אפליקציה על המחשב.

Mac:

  1. לך ל-claude.ai/download
  2. הורד את קובץ ה-.dmg
  3. פתח אותו וגרור את Claude לתיקיית Applications

Windows:

  1. לך ל-claude.ai/download
  2. הורד את ה-installer, לחץ Next עד הסוף

Windows בלבד: חובה להתקין Git

Claude Code משתמש ב-Git ברקע. בלי Git, Claude Code לא יעבוד על Windows. זה חינמי ולוקח דקה.

  1. לך ל-git-scm.com/download/win
  2. הורד והתקן (Next Next Finish)
  3. חשוב: וודא שהאפשרות "Add Git to PATH" מסומנת (היא מסומנת כברירת מחדל)

Mac לא צריך שום דבר נוסף.

אחרי ההתקנה, פתח את Claude Desktop בפעם הראשונה. הוא יבקש ממך להתחבר או ליצור חשבון. עקוב אחרי ההוראות על המסך.

💳 שלב 3: מנוי Pro ($20/חודש)

הגרסה החינמית מוגבלת מדי לצרכים שלנו. מנוי Pro פותח את כל מה שצריך באתגר: Skills, Scheduled Tasks, Connectors.

איך לשדרג:

  1. פתח את Claude Desktop
  2. לחץ על האייקון של הפרופיל שלך (פינה שמאלית תחתונה)
  3. בחר Upgrade או Subscribe
  4. בחר את תוכנית Pro ב-$20/חודש
  5. הכנס כרטיס אשראי ואשר

על הקרדיטים

תוכנית Pro מגיעה עם כמות מוגבלת של קרדיטים (שימושים). ככל שתשתמש יותר, תבחין שהם נגמרים מהר יותר. כשזה קורה, יש תוכנית Max ב-$100/חודש עם הרבה יותר קרדיטים. אבל לאתגר ולהתחלה, Pro מספיק.

מומלץ להשתמש במודל Sonnet כברירת מחדל כדי לחסוך קרדיטים. נסביר על המודלים בשלב 8.

🔀 שלב 4: שלושת המצבים (Chat, Cowork, Code)

💬
Chat

זה המקבילה של ChatGPT. שאלה, תשובה. בלי גישה לקבצים או כלים.

🤝
Cowork

ממשק נוח יותר לעבודה עם קבצים. Claude רואה ויכול לערוך. טוב לעריכת מסמכים.

⚡
Code

הטרמינל הידידותי. כאן Claude מריץ Skills, Scheduled Tasks, מתחבר לכלים. כאן דונה חיה. כאן אנחנו עובדים.

כלל: תמיד לעבוד מ-Code. לא Chat, לא Cowork. Code.

📁 שלב 5: Claude עובד עם תיקיות

זה קונספט חשוב שצריך להבין. Claude Code לא עובד "באוויר". הוא מתחבר לתיקייה ספציפית על המחשב שלך.

לפני כל סשן עבודה, אתה בוחר תיקייה (File > Open Folder). Claude קורא את הקבצים בתיקייה הזו, כותב אליה, ומשתמש בקובץ CLAUDE.md שנמצא בה.

זו הסיבה שדונה "חיה" בתיקייה ספציפית (donna-starter-kit). כשתפתח את התיקייה הזו ב-Claude Code, דונה תדע מי היא ומה היא יודעת לעשות. כשתפתח תיקייה אחרת, Claude לא יכיר את דונה.

בקיצור:

תיקייה = פרויקט. תיקייה אחרת = פרויקט אחר. CLAUDE.md בתיקייה = ההוראות שלך ל-Claude באותו פרויקט.

🗂️ שלב 6: הכרת הסיידבר של Claude Desktop

בצד שמאל של Claude Desktop (בטאב Code) יש סיידבר. כדאי להכיר מה יש בו:

New session - פותח שיחה חדשה. דונה לא זוכרת שיחות קודמות, אבל כן קוראת את CLAUDE.md מחדש. כל שיחה מתחילה "נקייה" עם הזיכרון מהקובץ.

Search - חיפוש בשיחות קודמות. שימושי כשאתה רוצה למצוא משהו שביקשת בעבר.

Scheduled - כאן רואים ומגדירים משימות מתוזמנות שרצות אוטומטית (נלמד ביום 5).

Customize - הגדרות, חיבור כלים חיצוניים (Connectors), והגדרות מודל. נכנס לזה בשלב הבא.

⚙️ שלב 7: הגדרות חשובות

כדאי להכיר את ההגדרות לפני שמתחילים לעבוד.

איך להגיע להגדרות:

לחצו על Customize בסיידבר (בצד שמאל), או על אייקון הפרופיל בפינה השמאלית התחתונה.

הגדרה חשובה: בחירת מודל

Claude מגיע עם כמה מודלים (גרסאות) שונים. המודל קובע כמה חכם וכמה קרדיטים הוא צורך. מומלץ להגדיר את Sonnet כברירת מחדל כדי לחסוך קרדיטים. נסביר על ההבדלים בשלב הבא.

את בחירת המודל אפשר לשנות גם מתוך שיחה: לחצו על שם המודל בפינה העליונה של חלון השיחה.

🤖 שלב 8: המודלים של Claude

Claude מגיע בכמה "גרסאות" שנקראות מודלים. כל מודל שונה ברמת החוכמה, המהירות, וכמות הקרדיטים שהוא צורך:

🐇
Haiku

מהיר, קל, זול. טוב למשימות פשוטות כמו תרגום, סיכום קצר, או שאלה מהירה.

⚡
Sonnet

מאוזן. מומלץ לאתגר. חכם מספיק לרוב המשימות, חוסך קרדיטים.

🧠
Opus

הכי חזק. צורך יותר קרדיטים. שמרו לניתוח מורכב או עבודה שדורשת חשיבה מעמיקה.

איך לשנות מודל:

הקלידו /model בתוך שיחה, או לחצו על שם המודל בפינה העליונה של חלון השיחה, או שנו בהגדרות (Customize).

כדי לחסוך קרדיטים, השתמשו ב-Sonnet כברירת מחדל.

Sonnet חכם מספיק ל-90% מהמשימות. תעברו ל-Opus רק כשצריכים ניתוח עמוק או כתיבה מורכבת.

🔐 שלב 9: מצבי אישורים

כשClaude רוצה לעשות משהו (לערוך קובץ, להריץ פקודה), הוא מבקש אישור. יש 4 מצבים:

✋ Ask permissions

Claude שואל לפני כל שינוי. הכי בטוח. טוב למתחילים שרוצים לראות מה קורה לפני שמאשרים.

</> Auto accept edits

Claude עורך קבצים בלי לשאול, אבל עדיין שואל לפני הרצת פקודות. אמצע הדרך.

📋 Plan mode

Claude מתכנן לפני שעושה. מציג תכנית ומחכה לאישור. טוב למשימות גדולות שרוצים לראות תכנית לפני ביצוע.

⚠️ Bypass permissions

Claude עושה הכל בלי לשאול. הכי מהיר. מומלץ למי שכבר מרגיש בנוח עם Claude ורוצה שהעבודה תזרום בלי עצירות.

המלצה לאתגר: תתחילו עם Auto accept edits. אחרי כמה ימים שתרגישו בנוח, עברו ל-Bypass permissions.

איפה משנים: לחצו על האייקון בפינה השמאלית התחתונה של שורת הקלט (ליד כפתור השליחה).

🔑 שלב 13: טיפ זהב. ייבוא זיכרון מ-ChatGPT

עוברים מ-ChatGPT? אל תתחילו מאפס. Claude יכול לייבא את הזיכרונות שצברתם ב-ChatGPT, כך שהוא כבר ידע דברים עליכם מהרגע הראשון.

איך עושים את זה:

  1. ב-ChatGPT: בקשו ממנו לייצא את הזיכרון שלו. הוא יפרט את כל מה שהוא זוכר עליכם.
  2. העתיקו את הטקסט שהוא מחזיר.
  3. ב-Claude Desktop: היכנסו להגדרות (Customize) > Import memory > הדביקו את הטקסט.

מדריך מפורט: לחצו כאן למדריך המלא

שימו לב: זו תכונה ניסיונית. היא עובדת הכי טוב עבור העדפות עבודה ומידע מקצועי, פחות עבור פרטים אישיים. לא חייבים לעשות את זה, אבל זה חוסך זמן.

🎓 שלב 14: טיפ זהב. פרומפט המאמן

יש טריק שגיליתי שהופך את Claude למאמן אישי. אתה מדביק פרומפט מיוחד בתחילת פרויקט, ו-Claude ממשיך לעבוד כרגיל, אבל מזמן לזמן זורק טיפים שעוזרים לך ללמוד.

למשל, אחרי שמסיימים משימה הוא עשוי להגיד: "כדאי לתעד את ההחלטה הזו. כדי שלא נבזבז שוב זמן על אותה שאלה." או אחרי שיחה ארוכה: "השיחה ארוכה, כדאי לדחוס אותה כדי לחסוך קונטקסט."

הוא לא מפסיק לעבוד ומתחיל ללמד. הוא עובד ומלמד במקביל. רק כשיש הזדמנות ללמידה ברורה.

🟣 שלב 16: התקנת Obsidian

Obsidian זו אפליקציה חינמית שמציגה קבצי טקסט בצורה ויזואלית. אנחנו משתמשים בה כדי לראות את הזיכרון של דונה. כשדונה כותבת סיכום, מתעדת החלטה, או שומרת מידע על אנשים, Obsidian מציג את הכל בצורה מסודרת עם קשרים ביניהם.

  1. לך ל-obsidian.md והורד (חינמי, Mac/Windows/Linux)
  2. התקן כמו כל אפליקציה

נחבר את Obsidian אחרי שנוריד את ה-Starter Kit בשלב הבא.

🔗 שלב 18: חיבור הכל

עכשיו מחברים את הכל: Claude Code + Obsidian, שניהם על אותה תיקייה.

ב-Claude Desktop:

  1. פתח Claude Desktop
  2. עבור לטאב Code
  3. File > Open Folder > בחר את donna-starter-kit
  4. בדוק שזה עובד:
What files do you see in this folder?

Claude צריך לראות: CLAUDE.md, brain/, templates/, guides/ ועוד.

ב-Obsidian:

  1. פתח Obsidian
  2. Open folder as vault > בחר את אותה תיקייה (donna-starter-kit)

🕐 3 המשימות המתוזמנות

Morning Briefing כל יום 9:15

מה עושה: סורקת מיילים, יומן ומשימות. מכינה סיכום בוקר עם סדרי עדיפויות.

מתי רץ: כל יום ב-9:15 אוטומטית.

מה תראה בבוקר: הודעה מדונה עם דוח מסודר. מיילים דחופים, פגישות היום, משימות פתוחות.

לאן הפלט נשמר: brain/Daily Logs/ ב-Obsidian

Evening Summary כל יום 16:00

מה עושה: סוגרת את היום + מכינה תדריכים לפגישות של מחר.

מתי רץ: כל יום ב-16:00 אוטומטית.

מה תראה: סיכום של מה שהושלם, מה נשאר פתוח, ותדריכי פגישות למחר.

לאן הפלט נשמר: brain/Daily Logs/ ב-Obsidian

Weekly Report ראשון 9:15

מה עושה: דוח שבועי. ניצחונות, פספוסים, מספרים, והמלצות לשבוע הבא.

מתי רץ: כל ראשון ב-9:15 אוטומטית.

מה תראה: מבט רחב על השבוע שעבר עם ניתוח מה עבד ומה לא.

לאן הפלט נשמר: brain/Daily Logs/ ב-Obsidian

חשוב: המחשב חייב להיות דלוק

Scheduled Tasks רצים רק כשהמחשב דלוק ו-Claude Desktop פתוח. אם המחשב כבוי, דונה לא תעבוד.

Fallback: אם המחשב היה כבוי, פשוט הרץ /morning-briefing ידנית כשפותח את המחשב. לוקח 5 שניות.

טיפ: תשאיר את המחשב דלוק בשעות שהגדרת (9:15 ו-16:00). או תריץ ידנית כשנוח לך.

❓ שאלות נפוצות

מה קורה אם המחשב כבוי?
דונה לא תעבוד. המשימות המתוזמנות רצות רק כשהמחשב דלוק ו-Claude Desktop פתוח. אם פספסת את הבריפינג של 9:15, פשוט הרץ /morning-briefing ידנית כשפותח את המחשב. לוקח 5 שניות והתוצאה זהה. אפשר גם לשנות את השעות ל-10:00 או כל שעה שנוחה לך.
אפשר להוסיף עוד Scheduled Tasks?
כן, בהחלט. כל סקיל יכול להפוך למתוזמן. אחרי שבוע שבועיים של שימוש, תזהה דפוסים: "כל שני אני צריך לבדוק מיילים נשכחים" או "כל חמישי אני מכין דוח ללקוח". תוסיף Scheduled Task חדש ודונה תעשה את זה בשבילך.
מה קורה אחרי האתגר?
ממשיכים. דונה ממשיכה לעבוד כל בוקר וערב. היא ממשיכה ללמוד. ככל שאתה משתמש בה יותר, היא נהיית יותר טובה כי brain/ מתמלא במידע. העלות היחידה היא מנוי Claude Pro ב-$20 לחודש. והקהילה ב-WhatsApp תמיד זמינה לשאלות.
אפשר להוסיף סקילס חדשים?
כן. אחרי האתגר, אתה יכול ליצור סקילס חדשים בעצמך. כל קובץ .md שתשים בתיקיית .claude/skills/ הופך לפקודה. תתחיל מלשכפל סקיל קיים ולשנות את ההנחיות. עם הזמן תבנה ספרייה שלמה מותאמת לעסק שלך.
הגרף ב-Obsidian ריק?
נורמלי לגמרי ביום 5. הגרף מתמלא ככל שדונה כותבת יותר קבצים ב-brain/. אחרי כמה ימים של בריפינגים יומיים, תדריכי פגישות, ומעקב משימות, יהיו שם עשרות נקודות מחוברות. בשבוע הבא תראה הבדל משמעותי.
