Step 2 . your gift

Install the /character skill

Paste this into your terminal, or straight into Claude Code on desktop. It installs globally, so it's available in every folder.

npx skills add JayGPTPro/character -g

This uses npx, which needs Node.js (it ships with Claude Code on most setups). No Node, or the command won't run? Skip it and paste this into Claude Code instead: "Install the skill from github.com/JayGPTPro/character into my global skills folder (~/.claude/skills), fetch the files yourself."

  1. Open Claude Code in the folder that is your employee. for example, a folder named "Michael", your creative director.
  2. Type /character and answer a few quick questions (role, how it talks to you, humor, energy, pushback).
  3. Say hi to your new employee. From now on, every chat in that folder is in character.

How it works, in plain words:

When you run it, the skill saves your AI's personality into a small file inside that folder. Every time you open the folder, your AI reads that file first, so it always remembers who it is.
