Tool 01 · A+ Content Engine
The Workflow · Genrupt MCP

A+ Content from a single ASIN.

Behind the scenes, the MCP does the heavy lifting. It researches the product, plans creative direction, structures the modules, writes the copy, and prepares the visual assets. After the one-time setup, your part is one line.

STEP 01

Drop the ASIN

Open Claude Code in any folder. Give it an Amazon ASIN (or the product URL) in the prompt. That's your entire input.

STEP 02

The MCP works

Claude Code calls the Genrupt MCP. It scrapes the listing, plans the modules, drafts the copy, and generates the imagery.

STEP 03

A+ and video out

The Setup Guide

5 steps. Set it up once.

Click any step to expand the exact install instructions. The hardest part is creating the Genrupt account. Everything else is paste-and-run.

1

Node.js 18+

Required for every install command on this page.

+

If you already use Claude Code, you almost certainly have Node.js installed. To check, paste this in Claude Code:

Check my Node.js version and tell me if I need to upgrade for MCPs.

If Claude tells you you need to install or upgrade, download the LTS installer from nodejs.org and run it. Works on macOS and Windows.

2

Claude Code Desktop

The app you will run the workflow from. macOS and Windows.

+

Step 1. Download Claude Code from claude.com/download and install it.

Step 2. Sign in with your Anthropic account. If you do not have one yet, create it at claude.ai first.

Step 3. Open Claude Code and create or open a project folder. This is where the workflow will run.

3

Create your Genrupt account

This is the engine behind the A+ generation. Account first, then the MCP and extension below.

+

Use the link below to sign up. The AI Vault community gets credited and you get the right onboarding.

→ Create your Genrupt account

Confirm your email before moving to the next step. The MCP needs a live account to authenticate.

4

Install the Genrupt MCP

Connects Claude Code to Genrupt so the workflow runs end-to-end.

+

Step 1. Go to genrupt.com/mcp and follow the install instructions there. Genrupt keeps that page up to date with the exact command for the current version.

Step 2. After install, restart Claude Code so it picks up the new MCP.

Step 3. Verify. Ask Claude:

List my MCP servers.

You should see genrupt in the list with a connected status.

5

Install the Genrupt Chrome extension

Lets Claude read the live Amazon page to ground the A+ in real product data.

+

Step 1. Open genrupt.com/extension and follow the install link to the Chrome Web Store.

Step 2. Sign into the extension with the same Genrupt account from Step 3.

Step 3. Pin the extension to your Chrome toolbar so it is one click away on any Amazon product page.

All three pieces (account, MCP, extension) are required for the workflow. If one is missing, the run will stop and tell you what to fix.

Run It

One prompt. One URL.

Once the setup is done, the workflow is the same every time. Open Claude Code, paste a prompt with any Amazon product URL, and let it run.

A+ Content + Product Video

Drop a prompt in Claude Code with the Genrupt MCP enabled. Give it the product's ASIN (or Amazon URL) right in the prompt. No need to open anything in Chrome.

Prompt 1 · A+ Content
Use the Genrupt MCP to generate 3 A+ content concepts for ASIN B0XXXXXXXX. Desktop only.
Prompt 2 · Product Video
Use the Genrupt MCP to generate a 15-second product video ad for ASIN B0XXXXXXXX with Seedance 2.0, 720p, fast mode.

These are just examples. Change 3 to any number of concepts, swap Desktop only for Mobile only, or add your own direction (tone, audience, modules). The MCP handles the rest. Each run usually costs only a few cents in credits.
