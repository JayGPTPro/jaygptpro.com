התקנת Git

Git זה הכלי שבעזרתו קלוד קוד שומר היסטוריה של כל מה שהוא משנה לכם בקבצים. בלי גיט אין אפשרות לחזור אחורה אם משהו השתבש. זה הביטוח שלכם.

סרטון הדרכה: התקנה על Windows
⊞ Windows
הורדת Git ל-Windows ←
⌘ Mac

במק Git כבר מותקן ברוב המקרים. כדי לוודא, פתחו את Terminal והקלידו git --version. אם לא מותקן, המערכת תציע להתקין את Xcode Command Line Tools. לחצו Install וזהו.

!
במהלך ההתקנה ב-Windows, פשוט לחצו Next בכל החלונות. אין צורך לשנות שום הגדרה.
!
חשוב לפני שעוברים לשלב 2
אחרי שהתקנתם את Git, כבו והדליקו את המחשב לפני שתפתחו את Claude. זה נשמע מוזר, אבל זה הצעד שפותר את רוב הבעיות שאנשים נתקלים בהן בהמשך. אל תדלגו עליו.
2

התקנת Claude למחשב

זה השחקן הראשי. Claude Desktop זה האפליקציה שדרכה תעבדו עם Claude Code. תורידו, תתקינו, ותתחברו עם החשבון שלכם (אם אין לכם, תיצרו עכשיו, חינם).

סרטון הדרכה: התקנה על Windows
⊞ Windows + Mac
הורדת Claude Desktop ←

הלינק מזהה אוטומטית באיזו מערכת אתם ומציע את הקובץ הנכון. במק: תורידו .dmg, תפתחו, תגררו את Claude לתיקיית Applications. בווינדוס: תורידו .exe ותריצו.

!
אחרי ההתקנה, פתחו את Claude ותתחברו עם המייל שלכם. אין לכם חשבון בכלל? צרו עכשיו (חינמי) ואז שדרגו ל-Pro של 20$ לחודש. בלי Pro אי אפשר להשתמש ב-Code tab ולא תוכלו לעקוב אחרי האתגר. זו ההוצאה היחידה.
3

התקנת התוסף לכרום

תוסף Claude in Chrome מאפשר ל-Claude לעבוד ישירות בדפדפן שלכם. לקרוא דפים, ללחוץ על כפתורים, למלא טפסים. בלעדיו, לא תוכלו להריץ חצי מהדברים שנעשה באתגר.

סרטון הדרכה: התקנה על Windows (זהה במק)
⊞ זהה בכל המערכות
התקנת התוסף מ-Chrome Store ←

התוסף עובד אך ורק על Google Chrome. אם אתם משתמשים ב-Safari, ב-Edge או בדפדפן אחר, תצטרכו להוריד גם את Chrome. הוא חינמי וזה ייקח עוד דקה.

!
אחרי ההתקנה, לחצו על אייקון התוסף בכרום, תתחברו עם החשבון שלכם, ותאשרו את ההרשאות שהוא מבקש. בלי האישור התוסף לא יוכל לעבוד.

שאלות נפוצות

אם נתקלתם בבעיה, כנראה שמישהו כבר נתקל בה לפניכם.

כן. הוובינר הוא הדגמה חיה, ואם תרצו להריץ במקביל את מה שאני מראה, אתם חייבים את הכלים מותקנים. גם אם אתם רק מסתכלים, מומלץ שיהיה הכל מוכן כי זה החלק הכי משעמם והכי טכני, ועדיף להוריד מעליכם את זה לפני שמתחילים.
הסרטונים מראים את התהליך בווינדוס כי הוא קצת יותר מורכב. במק זה הרבה יותר פשוט, ולכן הספקנו זאת בכמה שורות טקסט. עברו לפי ההוראות בכל קלף ויהיה לכם הכל תוך 5 דקות.
אם הלינק לא עובד, היכנסו ידנית ל-git-scm.com, לחצו על Downloads, ובחרו את מערכת ההפעלה שלכם. ההורדה תתחיל אוטומטית. אם עדיין לא, ייתכן שיש לכם חוסם פרסומות (Adblock) שחוסם את הסקריפט. כבו אותו לרגע.
Claude Pro של 20$ לחודש זה החובה. בלעדיו אין גישה ל-Code tab, ל-Skills ול-Scheduled Tasks. כל האתגר מבוסס על הפיצ'רים האלה. גם אחרי האתגר, ה-20$ האלה יחזירו את עצמם שורה תחתונה תוך שעת עבודה אחת בחודש.
כן, זה בטוח. התוסף הזה רשמי של Anthropic (החברה שמאחורי Claude). ההרשאות הרחבות נחוצות כי התוסף צריך לראות מה יש בדפים שאתם פותחים, וללחוץ עליהם בשמכם. זה בדיוק מה שעושה אותו כל כך חזק. אם זה לא נוח לכם, אל תתקינו, אבל אז לא תוכלו להשתמש ביכולות של דפדפן באתגר.
