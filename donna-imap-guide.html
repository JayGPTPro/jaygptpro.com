Add a Second Email to Donna

Two steps. You get a password, Donna does the rest.

01
Get an App Password from Google
The one thing only you can do — Google needs to confirm it's you.
Go to myaccount.google.com/apppasswords (requires 2FA to be active).
Type any name (e.g. Donna) → click Create
Copy the 16-character code that appears
02
Paste this to Donna — she handles everything else
Open Claude Code and send her this. Replace the two green parts.
Prompt to paste 
I want to give you access to my second email account:
[your@gmail.com]

My Google App Password for this account is:
[paste 16 chars here]

Please:
1. Build a Python IMAP client that lets you read,
   search, and send emails from this account.
   Save it to ~/Downloads/Claude/donna-imap/imap.py
2. Test it — show me the last 5 emails.
3. Add a section to my global CLAUDE.md so you
   remember this account in every future session.
✓
That's it.
Donna builds the script, tests it, and updates her own memory. From now on, in any session, in any folder — she knows about the second email.
01
Get an App Password from Microsoft
Go to account.microsoft.com/security → Advanced security options → App passwords (requires 2FA to be active).
Click Create a new app password
Copy the password that appears
Work / company email? Some Microsoft 365 Business accounts block this at the IT level. If it doesn't work, check with your IT team.
