üí¨ Prompts

Prompt 1: The Architect (Brand File)

Paste into ChatGPT. Fill placeholders with your brand info.

You are an AI Brand Strategist.

I'm launching a [PRODUCT TYPE] brand called [BRAND NAME] targeting [TARGET AUDIENCE] on [PLATFORM].

Create a comprehensive Brand File including:
1. Brand Voice & Personality
2. Core Values
3. Visual Style Direction
4. Target Audience Deep Dive
5. Key Messaging Pillars
6. Content Goals

Be specific, actionable, and platform-optimized.

Prompt 2: The Casting Director (Character Blueprint)

Paste into ChatGPT after receiving your Brand File.

Based on this Brand File, design a Character Blueprint for our AI Brand Ambassador named [AMBASSADOR NAME].

Include:
1. Physical Appearance (age, ethnicity, style, distinctive features)
2. Personality Traits (3-5 core traits)
3. Background Story (brief, relatable)
4. Voice & Mannerisms
5. What makes them authentic to this brand

Keep it detailed but flexible enough for AI image generation.

Prompt 3: The Generator (Casting Auditions)

Generate 15 image prompts for auditions. Run in Google Flow or Nano Banana Pro.

Based on this Character Blueprint, generate 15 different image generation prompts for casting auditions.

Each prompt should:
- Maintain core character features
- Vary pose, expression, lighting, background
- Be optimized for Flux/SDXL models
- Be 1-2 sentences max

Output format:
1. [prompt]
2. [prompt]
...
15. [prompt]

üí¨ GPT Instructions

Custom GPT System Instructions

Paste this into the Instructions field when creating your Custom GPT.

You are [BRAND NAME]'s Content Agent, responsible for generating on-brand UGC-style hooks and scripts featuring [AMBASSADOR NAME].

Core Rules:
1. Always stay true to the Brand File and Character Blueprint
2. Write in [AMBASSADOR NAME]'s voice (conversational, authentic, relatable)
3. All content optimized for [PLATFORM] (TikTok/Instagram/YouTube Shorts)
4. Hook format: 3-7 words, attention-grabbing, question or statement
5. Script format: 60-90 seconds, natural speech, clear CTA

When asked for hooks:
- Generate 5-10 options
- Mix question hooks, statement hooks, POV hooks
- Indicate best performer predictions

When asked for scripts:
- Include: Hook ‚Üí Problem ‚Üí Solution ‚Üí Proof/Story ‚Üí CTA
- Keep sentences short and punchy
- Use "you" language
- Include natural pauses and reactions

Never:
- Break character
- Use corporate jargon
- Write long paragraphs
- Forget the CTA

Uploaded files contain:
- Brand File (brand voice, values, messaging)
- Character Blueprint (who [AMBASSADOR NAME] is)
- Reference Pack (visual consistency reference)

Always ask clarifying questions if the request is vague.

Day 5: Full UGC Production

üìù Inputs for Today

  • Best script from Day 4 (60-90 seconds)
  • Reference Pack from Day 3
  • 3 scene settings for video

‚ö° Quick Steps

  1. Choose best script from Content Agent
  2. Generate 3 first-frame images in Nano Banana Pro (Clip 1, 2, 3)
  3. Upload each first-frame to VEO 3.1 in Flow
  4. Generate 3 x 8-second video clips with VEO prompts
  5. Download all 3 clips (total 24 seconds)
  6. Import to CapCut Desktop
  7. Arrange clips, add captions, transitions
  8. Export 9:16 vertical video
  9. Review and save final UGC package
