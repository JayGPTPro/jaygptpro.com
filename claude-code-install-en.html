Install Git

⏱ About 3 minutes

Git is a version control tool that Claude Code uses under the hood to manage files. Mac users usually already have it. Windows users must install it.

✓ Most Mac users can skip this step entirely. Git comes pre-installed on macOS. Move on to Step 2. Only come back here if Step 2 fails.

If you run into issues, here's how to check and install Git:

  • 1Open Terminal (press Cmd + Space, type "Terminal", press Enter).
  • 2Type git --version and press Enter. If you see a version number, you're good.
  • 3If a popup appears asking to install developer tools, click Install. Takes about 2 minutes. Then proceed to Step 2.
!
Windows users: this step is required.
Claude Code on Windows requires Git to be installed and added to your system PATH. Without this, Claude Code will not run at all. Follow the steps carefully.
  • 1Go to git-scm.com/download/win and download the installer.
  • 2Run the installer. When you reach the screen that says "Adjusting your PATH environment", choose "Git from the command line and also from 3rd-party software". This is critical.
  • 3Keep all other options as default and click Next through to the end. Click Install.
  • 4Restart your computer now. This makes sure Windows picks up the new Git installation. It takes 1 minute and prevents most problems.
i If Claude Code still shows an error after installing Git, a restart is almost always the fix. The PATH change requires it.

Install Claude Code Desktop

⏱ About 5 minutes

Claude Code is the tool. You need a Claude Pro subscription ($20/month) to use it. The free plan does not include Claude Code.

  • 1Go to claude.ai and sign in (or create a free account first).
  • 2Upgrade to Claude Pro for $20/month. Click your profile icon → Upgrade Plan.
  • 3Once on Pro, go to claude.ai/download and download the Mac version.
  • 4Open the downloaded file and drag Claude.app to your Applications folder.
  • 5Open Claude from your Applications. Sign in with your account. You should see three tabs at the top: Chat, Cowork, Code.
  • 6Click the Code tab. This is Claude Code. You're in.

3

Install the Chrome Extension

⏱ About 1 minute

This extension lets Claude Code see and control your browser. We'll use it during the demos. Works on Mac and Windows, same steps.

  • 1Make sure you have Google Chrome installed. If not, download it first from google.com/chrome.
  • 2Open Claude Code Desktop. Go to the Code tab.
  • 3Type this in the chat: install chrome extension and press Enter. Claude will give you the direct link.
  • 4Click the link, then click Add to Chrome on the Chrome Web Store page. Confirm when prompted.
  • 5Once installed, you'll see the Claude icon appear in your Chrome toolbar. You're done.
i If you don't see the icon after installing, click the puzzle piece 🧩 icon in Chrome's toolbar and pin the Claude extension so it stays visible.
