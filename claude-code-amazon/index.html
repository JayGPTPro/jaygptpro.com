Alexa Listing Audit

In-browser Q&A mining + listing gap analysis
Claude for Chrome
What it does
Uses Claude's Chrome extension to query Amazon's "Alexa for Shopping" panel on a product page. Captures the top customer questions Alexa surfaces, then cross-references them against your current title and bullets to identify exactly which buyer concerns are unanswered in your listing.
What you need
✓
Claude for Chrome installedView install
How to run it
  1. Open your Amazon product page in Chrome
  2. Activate the Claude for Chrome extension on this tab
  3. Paste prompt 1, wait for the Q&A table
  4. Then paste prompt 2 for the listing gap analysis
Prompt 1. Pull Q&A from Alexa
On the current Amazon tab, open the "Alexa for Shopping" panel and type this question into its chat box. "What questions do customers most commonly ask before buying this product?" Submit it, wait for Alexa's full answer, then return the top 10 Q&A pairs as a clean table with Question and Answer columns. Do not click any of the suggested question buttons.
Prompt 2. Map to listing
Now check if these answers already appear in the listing title or bullets. What's missing and how should I fix it? Mark each as ✅ answered, ⚠️ partial, or ❌ missing.

SQP Funnel Diagnostic

Defend / Growth / Kill / Monitor classification of every keyword
Brand Analytics CSV download
What it does
Analyzes the Search Query Performance export from Brand Analytics. Classifies every search query into one of four strategic buckets, ranks top priority actions for the week, and surfaces hidden insights most sellers miss. Saves the full report to your filesystem and updates a tasks checklist.
What you need
✓
Amazon Brand Registry
How to download the SQP report
  1. Seller Central → Brands → Brand Analytics → Search Analytics → Search Query Performance
  2. Choose Brand, Reporting Range = Weekly, Select Week (last 4 weeks recommended)
  3. Click Apply
  4. Top right: Generate Download → choose Comprehensive View (all 33 columns)
  5. Wait for processing to finish. Track status in Download Manager
  6. Download the CSV when ready
How to run it in Claude Code
  1. Open Claude Code in your project folder
  2. Drag the SQP CSV into the chat
  3. Paste the prompt below
Prompt
Analyze this SQP report for my Amazon brand. Start with a 3-sentence bottom line covering the state of my keyword portfolio, the biggest opportunity I'm missing, and the biggest risk this week. Then classify every search query into one of four buckets: Defend (I dominate, maintain, including branded searches where I should flag any impression share leakage to competitors), Growth (good CTR and CVR but low share, increase PPC), Kill (clicks but no conversion, OR many impressions but few clicks, either way a listing or image bottleneck), or Monitor (no meaningful brand exposure, skip for now). Compute CTR Index as brand clicks divided by brand impressions, vs the category equivalent (not the search-volume funnel rate). Same logic for CVR. Sort by search volume. Then give me the top 5 priority actions for this week, each with the exact keyword, why it's a priority, the specific action to take, and a realistic estimated dollar impact. Finish with one strategic surprise, something hidden in this data that most sellers would miss. Be specific and use the actual numbers. When you're done, save the full report as reports/sqp-week-[current week number].md and add the 5 priority actions to tasks.md as a checklist.

Market Basket Analysis

Bundle opportunities + customer identity insights
Brand Analytics CSV download
What it does
Analyzes the Market Basket Analysis export. Identifies products customers buy together with yours, classifies each pair as Bundle Ready / SD Target / Complement / Noise, recommends the top monthly priority actions, and reveals what the basket says about who your customer actually is.
What you need
✓
Amazon Brand Registry
How to download the Market Basket report
  1. Seller Central → Brands → Brand Analytics → Consumer Behavior Analytics → Market Basket Analysis
  2. Choose Reporting Range = Monthly (more data than Weekly), select prior month
  3. Click Apply
  4. Generate Download → Comprehensive View → CSV
Prompt
Analyze this Market Basket report. Check the reporting date first, if older than 6 months, flag at the top that the data is stale and recommend pulling a fresher report before acting. Show top co-purchase pairs sorted by attach rate, classified as Bundle, SD Target, Complement, or Noise (use Noise for unrelated items and for variant cross-references that inflate false signals). Top 3 actions for this month with realistic dollar impact. End with one hidden insight about who my customer really is. Save to reports/market-basket-[period].md and update tasks.md.

Marketing Gold Mining

Category-wide competitive intelligence via Bright Data
Bright Data MCP
What it does
Scans the first 3 pages of Amazon search results for a keyword (~48 listings). Deep-scrapes every product page for titles, bullets, and image alt text. Mines for marketing gold, bonus items, unique features, memorable hooks, trust signals, and unusual positioning. Outputs a "steal this first" priority list of the top 5 ideas to test.
What you need
✓
Bright Data MCP installedView install
How to run it
  1. Open Claude Code
  2. Replace [KEYWORD] with your category keyword (e.g., "karaoke machine")
  3. Paste the prompt
  4. Let it run. Around 51 Bright Data calls total, well within the free tier.
Prompt
Use Bright Data's search_engine for [KEYWORD] site:amazon.com across the first 3 search pages. Then deep-scrape every product page returned to capture title, all bullets, and image alt text. Now mine this data for marketing gold, anything a competitor is doing that I could steal or adapt. Specifically look for bonus items included (free X, comes with Y), unique features no one else mentions, memorable marketing hooks or phrases, trust signals (certifications, awards, made in USA), and unusual product positioning. For each gold nugget give me the source ASIN, the exact text or feature, and why it works. Group findings by category: Bonuses, Hooks, Features, Trust Signals, Positioning. End with a "steal this first" priority list of the top 5 ideas I should test on my listing this week. Save the report as reports/marketing-gold-[keyword].md.

49 More Amazon Skills

The full nexscope-ai/Amazon-Skills repository, covering pricing, advertising, listing optimization, product research, analytics, competitive analysis, growth, and operations. Free, MIT licensed, and installed with one command. Already part of Setup Step 1 above. If you skipped it then, run it now and every skill below becomes available immediately in Claude Code.

Install

Run this once in your terminal. It clones the repo, installs all 49 skills globally to ~/.agents/skills/, and symlinks them into Claude Code's skills folder. After install, every skill becomes available in every Claude Code session. 

npx skills add nexscope-ai/Amazon-Skills -g

Verify on macOS / Linux: ls ~/.agents/skills/ | grep amazon

Verify on Windows (PowerShell): Get-ChildItem "$env:USERPROFILE\.agents\skills" | Select-String amazon

To use any of them, just ask Claude Code naturally, for example, "Use the amazon-keyword-research skill to research 'wireless karaoke microphones'".
