Import Your Memory Into Claude

Switching from ChatGPT? Don't start from zero. Move everything it learned about you. In 3 steps.

📤 Export from old AI
📋 Copy
📥 Import to Claude
1

Export your memory from ChatGPT (or any AI)

Go to ChatGPT (or Gemini, or any other AI) and paste this exact prompt. It's designed to pull everything it knows about you in one shot:

I'm moving to another service and need to export my data. List every memory you have stored about me, as well as any context you've learned about me from past conversations. Output everything in a single code block so I can easily copy it. Format each entry as: [date saved, if available] - memory content. Make sure to cover all of the following. Preserve my words verbatim where possible: Instructions I've given you about how to respond (tone, format, style, 'always do X', 'never do Y'). Personal details: name, location, job, family, interests. Projects, goals, and recurring topics. Tools, languages, and frameworks I use. Preferences and corrections I've made to your behavior. Any other stored context not covered above. Do not summarize, group, or omit any entries. After the code block, confirm whether that is the complete set or if any remain.

You'll get a text block with everything it knows. Copy all of it.

2

Open the import flow in Claude

Two ways to get there:

Option A: Go to Settings → Capabilities → Memory → Start Import

Option B: On Claude's home screen, click the "Import memory to Claude" card

Claude's Import Memory dialog. This is where you paste the text from Step 1
3

Paste and import

Paste the text you copied in Step 1 into the text box and click "Add to memory".

Claude will extract the key information and save it as individual memory entries. You can review what it saved by clicking "Manage edits".

✓

Verify it worked Optional

Once the import is done, open a new chat and ask:

What do you know about me?

If Claude responds with your preferences, work context, and style, you're all set. If something's missing, you can always add it manually via Settings → Capabilities → View and edit your memory.

⚠️ Good to know

  • This feature is still experimental and in active development
  • Claude focuses on work-related context: your preferences, style, tools, and instructions transfer best
  • Personal details unrelated to work may not always stick
  • You can review and edit imported memories anytime in Settings

💡 Pro tips

  • If your AI supports file generation, ask it to export your memory as a Markdown file so you keep a backup copy
  • You can edit the text before importing. Remove anything sensitive or irrelevant
  • For Amazon sellers: make sure your brand voice, listing style rules, and product knowledge are in there. That's what makes Claude immediately useful
