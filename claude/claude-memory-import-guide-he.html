ייצא את הזיכרון מ-ChatGPT (או כל AI אחר)

היכנסו ל-ChatGPT, ל-Gemini, או לכל AI אחר שהשתמשתם בו, והדביקו את הפרומפט הבא. הוא מתוכנן לשלוף הכל מה שהוא יודע עליכם בפעולה אחת:

🔒 המידע נשאר בחשבון שלכם בלבד. הפרומפט מייצא מה-AI הישן אל הטקסט שלכם. אתם בוחרים מה לייבא ומה להשאיר בחוץ.
prompt 
I'm moving to another service and need to export my data. List every memory you have stored about me, as well as any context you've learned about me from past conversations. Output everything in a single code block so I can easily copy it. Format each entry as: [date saved, if available] - memory content. Make sure to cover all of the following. Preserve my words verbatim where possible: Instructions I've given you about how to respond (tone, format, style, 'always do X', 'never do Y'). Personal details: name, location, job, family, interests. Projects, goals, and recurring topics. Tools, languages, and frameworks I use. Preferences and corrections I've made to your behavior. Any other stored context not covered above. Do not summarize, group, or omit any entries. After the code block, confirm whether that is the complete set or if any remain.

תקבלו בלוק טקסט עם כל מה שה-AI יודע עליכם. העתיקו הכל.

2

פתחו את תהליך הייבוא בקלוד

יש שתי דרכים להגיע לשם:

אפשרות א: לכו להגדרות דרך Settings › Capabilities › Memory › Start Import

אפשרות ב: במסך הבית של קלוד, לחצו על הכרטיסייה "Import memory to Claude"

תיבת ייבוא הזיכרון בקלוד. כאן מדביקים את הטקסט משלב 1
3

הדבק וייבא

הדביקו את הטקסט שהעתקתם בשלב 1 לתוך תיבת הטקסט ולחצו על "Add to memory".

קלוד ישלוף את המידע הרלוונטי וישמור אותו כרשומות זיכרון נפרדות. ניתן לבדוק מה נשמר על ידי לחיצה על "Manage edits".

✓

ודא שזה עבד אופציונלי

כדאי לבדוק, כי לפעמים קלוד שומר את המידע אבל לא בדיוק שולף אותו נכון. שאלה אחת בשיחה חדשה תגלה את זה מיד:

prompt 
What do you know about me?

אם קלוד עונה עם ההעדפות שלכם, הקשר המקצועי, וסגנון העבודה שלכם, הכל עבד. אם משהו חסר, תמיד אפשר להוסיף ידנית דרך Settings › Capabilities › View and edit your memory.

💡 טיפים מקצועיים

  • אם ה-AI שלכם תומך ביצירת קבצים, בקשו ממנו לייצא את הזיכרון כקובץ Markdown לגיבוי
  • ניתן לערוך את הטקסט לפני הייבוא. הסירו כל מידע רגיש או לא רלוונטי
