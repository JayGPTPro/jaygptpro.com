How it works

Connect once. Then just ask.

Genrupt is an e-commerce content engine. The skills teach Claude Code how to drive it, and the MCP gives Claude the tools to do the work. After a one-time setup, your part is one sentence in chat.

Install

Two commands. One restart.

This is the whole setup. Create your account, run two commands in the terminal, restart Claude Code, and the skills appear. The only real step is the account.

1

Create your Genrupt account

This is the engine behind every skill. Sign up with the link below, then confirm your email.

Create your Genrupt account →

2

Install the Genrupt CLI

Open your terminal and paste this. It installs the Genrupt command-line tool.

npm install -g @genrupt/cli@latest
3

Connect everything to Claude Code

Then run this one command. A browser window opens to sign you in to Genrupt, then it connects the Genrupt MCP to Claude Code and installs the six skills, all at once. No custom connector to set up by hand.

genrupt agent install
4

Restart Claude Code

Fully quit Claude Code and open it again. This is what loads the new skills and tools. Skipping it is the number one reason people do not see the skills.

5

Verify it worked

Ask Claude this. You should see the six genrupt-* skills and the Genrupt MCP listed.

List my skills and MCP servers.

You signed in during the previous step, so you are ready to go. If Claude Code ever asks you to re-approve the Genrupt connection, approve it once and you are set.

Before you start: prerequisites +

You need two things installed first. If you already use Claude Code, you have both.

  1. Claude Code from claude.com/download, signed in with your Anthropic account.
  2. Node.js 18 or newer, which powers the install commands. Not sure? Ask Claude: Check my Node.js version and tell me if I need to upgrade. If you need it, get the LTS installer at nodejs.org. Works on macOS and Windows.
Using the Claude desktop app instead of Claude Code? +

The two commands above are for Claude Code. If you want Genrupt in the Claude desktop chat app, add it as a custom connector instead:

  1. Open Claude, go to Settings → Connectors.
  2. Click Add custom connector, name it Genrupt, and paste this URL:
https://genrupt.com/api/agent/mcp

Click Add → Connect and sign in with your Genrupt account.

FAQ

Quick answers.

Do I need to know how to code? +
No. After the one-time setup, everything happens by asking Claude Code in plain English. The two install commands are copy and paste. You never write code.
What is the difference between the skills and the MCP? +
The MCP gives Claude the Genrupt tools, the actual buttons it can press. The skills teach Claude how and when to use them, so it walks you through a plan, asks the right questions, and waits for your approval. The genrupt agent install command sets up both at once.
What does it cost to run? +
Each generation spends a few Genrupt credits, and the skill shows you the plan before it spends anything. Your Claude Code subscription covers the chat itself. Code JAY20 adds 20 credits to your first order to get you started.

Mac or Windows? +
Both. Claude Code, Node.js, and the Genrupt CLI all run on macOS and Windows. The install commands are identical on each.
Can I edit what Genrupt generates? +
Yes. You review every plan before it generates, you can ask for edits on A+ concepts and video frames, and A+ exports include a layered PSD so your designer can take it further.
I ran the commands but do not see the skills. +
