Step 1 · Connect

Connect Genrupt to your agent.

Add Genrupt as a custom MCP connector. That gives your agent the Genrupt tools, in Claude, Codex, Cursor, Perplexity, and more.

New to Genrupt? Create your account first. Code JAY20 adds 20% extra credits to your first order. Create account →
Claude Codex Cursor OpenClaw Hermes NemoClaw Perplexity
1

Open connector settings

Desktop app: File → Settings → Connectors → Customize.
Browser: Settings → Connectors.

2

Add a custom connector

Click +, choose Add custom connector, name it Genrupt, and paste the MCP server URL:

https://genrupt.com/api/agent/mcp
3

Connect and sign in

Click Add → Connect and sign in with your Genrupt account. Your agent can now reach Genrupt.

Connection trouble? +

Sign in to Genrupt and reset the stored access for your agent. If you switched Google accounts, reset while signed in to the account you connected first. Then remove and re-add Genrupt in your connector settings and start a fresh chat.

Step 2 · Skills

Add the workflow skills.

The connector gives your agent Genrupt's tools. The skills teach it the workflows. Install them from your terminal, and re-run the same command anytime to update.

1

Run two commands in your terminal

This installs the Genrupt CLI, then drops the six workflow skills into your agent. Run the same command again anytime to update them.

npm install -g @genrupt/cli@latest
genrupt agent install
2

Restart, then verify

Fully quit and reopen your agent so it loads the new skills. Then ask it this. You should see the six genrupt-* skills.

List my skills.
Before you start: prerequisites +

You need two things installed first. If you already use Claude Code, you have both.

  1. Claude Code from claude.com/download, signed in with your Anthropic account.
  2. Node.js 18 or newer, which powers the install commands. Not sure? Ask Claude: Check my Node.js version and tell me if I need to upgrade. If you need it, get the LTS installer at nodejs.org. Works on macOS and Windows.
